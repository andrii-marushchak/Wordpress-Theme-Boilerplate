### Run Server

Set **webpackModules** if you want to use ES6 imports
```js
const webpackModules = true;
```

```bash
npm i
gulp
```


### Run Watcher

```bash
gulp
```

### Generate .pot file

```bash
gulp pot
```

### Compile .SCSS

```bash
gulp scss
```

### Compile .JS

```bash
gulp js
```

### Compress images

```bash
gulp img
```
